# ManUnitedApi - GO

### An Api to search Manchester United's players made in Go Lang

### Build
```
go build
```

### Execute
If build just execute the binary. If not, just run:
```
go run main.go
```
### Use
In browser:
```
To get all Players: http://localhost:3000/
To get one Player: http://localhost:3000/player/name
```
